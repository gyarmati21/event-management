# Event Manager (PHP Laravel)
 ### Prerequisites: 
	-   PHP 8.x
	-   Composer
	-   MySQL
	-   Laravel 9.x (or higher)

##  Setup
Before serving, you will need to do a `npm run dev`, without this the UI will not work.

	- composer install
	- npm install
	- php artisan migrate --seed
	- npm run dev
	- php artisan serve
